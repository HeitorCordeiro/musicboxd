Bem-vindo ao musicboxd

Um sistema orientado a objetos em C++ para catalogar músicas e avaliações, inspirado no Letterboxd.

Sobre o Projeto

Este projeto é um sistema de informação em C++ inspirado no Letterboxd, focado em música. Permite aos usuários catalogar músicas, artistas e escrever avaliações, criando uma pequena rede social de descoberta musical.

Funcionalidades Principais

  • Cadastro de Usuários: Diferenciação entre Administradores (com poder de moderação) e Usuários Comuns.
  • Sistema Social: Permite seguir usuários, avaliar músicas, visualizar outras avaliações e curtir destaques.
  • Moderação: Administradores podem suspender usuários, gerenciar músicas e ocultar/revelar avaliações.
  • Persistência de Dados: O estado do sistema é salvo em arquivos `system.json` e `users.db` e recarregado ao iniciar.

Resumo das Classes Principais

Sistema

Classe central (Façade) que gerencia todos os vetores de dados (usuários, músicas) e controla a persistência.

Usuario

Armazena dados de login, senha, e as listas de seguidores, avaliações criadas e status (admin/ativo/suspenso).

Artista

Entidade que armazena o nome e gênero, além de uma lista de ponteiros para suas músicas cadastradas.

Musica

Armazena os dados da música (título, álbum) e um ponteiro para seu Artista. Gerencia o vetor de avaliações recebidas.

Avaliacao

Classe base para avaliações. Contém nota, comentário, ponteiro para o autor e status de moderação (oculto).

AvaliacaoComDestaque

Classe derivada (Herança) que adiciona a funcionalidade de "curtidas" (um vetor de emails) a uma avaliação.

Como Executar o Projeto

Modo Recomendado (CMake)

mkdir -p build
cd build
cmake ..
make -j4
./musicboxd

Modo Rápido (g++)

g++ -std=c++17 -Wall -Wextra \
    src/main.cpp \
    src/models/*.cpp \
    -o musicboxd -Isrc/
./musicboxd
